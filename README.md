# Competitor Social Media Monitor Prototype

A lightweight prototype that automatically generates structured competitive intelligence reports from social media data using DeepSeek API + Python.

## Features

- Mock data ingestion (easily replaceable with real API)
- LLM-powered analysis via DeepSeek API
- Structured Chinese briefing: summary, key topics, market actions, sentiment

## Sample Output (Tested with @OpenAI)

**Core Summary**: OpenAI announced GPT-4o, a multimodal model accepting text/audio/image input; partnered with Financial Times for real-time journalism integration.

**Key Topics**: Multimodal model release / High-quality content partnership

**Market Actions**: GPT-4o launch / FT strategic collaboration

**Sentiment**: Positive, forward-looking
